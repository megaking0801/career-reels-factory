# 第 1 支｜給 Kling 用的素材與操作

## 一次性準備：用 ChatGPT 生「固定角色圖」（你帳號的臉，之後每支都用同一張）
把下面這段貼到 ChatGPT（生圖）：

```
請畫一張寫實風格的人物半身照，當作短影音帳號的固定虛擬主持人：
一位 30 歲出頭的台灣職場前輩，幹練、親切但帶點犀利感，
深色簡約襯衫，自然光，乾淨的辦公室或都市街道背景（淺景深），
正面看鏡頭，表情自信。直式 9:16 構圖，臉部清晰、光線均勻。
請固定這個長相，之後我會用同一個人物產系列影片。
```
> 生出來滿意就存檔，命名例如 `角色_前輩.png`，**以後每支都用這張**（換臉觀眾就記不住了）。

## 這一支的內容

**口播逐字稿（要餵給 Kling 的）**　← 同資料夾 `口播語音_第1支.mp3` 已是這段的語音
```
你以為面試官會主動跟你聊薪水跟未來發展？做夢吧
面試結束前，記得問面試官『這個職位最想看到的成就跟表現是什麼』

面試不是考試，別只是答題，要把握主動權。別忘了追蹤我們，學更多實用求職技巧
```

**畫面建議（給你參考，不會進影片）**

## Kling 操作
1. 進 Kling → 找「對口型 / AI Avatar / Lip Sync」
2. 上傳你的**角色圖**
3. 聲音：**上傳這個資料夾的 `口播語音_第1支.mp3`**（已用台灣男聲做好）
   - 或不用我的檔，直接在 Kling 貼上面的逐字稿、選中文（國語）語音也可以
4. （可選）先用圖生影片讓他有走動/運鏡，再對口型
5. 設定：**直式 9:16**、長度要能放完整段（約 28 秒）
6. 生成 → **下載 mp4**，下載前看角落有沒有浮水印

## 下載後
把 mp4 上傳回你的工具（http://localhost:8000 那張任務卡），
工具會自動疊提問字卡 + 上字幕 + 轉直式 → 下載成品 → 發 IG。
