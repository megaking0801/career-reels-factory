② 製作中 / 成果

🧭 完整流程（一支影片從 0 到發 IG）

  1. （一次性）做角色圖：用 ChatGPT 生一張固定的「虛擬前輩」臉，之後每支都用同一張。prompt 見下方，可一鍵複製。
  2. 產腳本：到「🎬 製作」分頁，選內容線/主題（都可留空 → AI 自動出題）→ 產生腳本。
  3. 複製口播：腳本卡按「📋 複製口播逐字稿」。
  4. Kling 做影片：用「角色圖 + 口播語音」做對口型 mp4、下載（步驟見下）。
  5. 上傳回來：在腳本卡上傳 mp4 → 工具自動上字幕＋疊開場問題字卡＋轉 9:16。
  6. 下載成品 → 手動發 IG Reels。

雙聲設計：avatar 只負責「前輩回答」；「藏鏡人提問」是工具合成時自動疊上的字卡，所以要上傳回工具才看得到完整成品。

🧑‍💼 角色圖 prompt（貼到 ChatGPT 生圖）

生出來臉太小或側臉，就跟 ChatGPT 說「臉再大一點、更正面、頭到胸口」重生。滿意就存檔，以後每支都用同一張，別換臉。

🎭 Kling 操作步驟（最簡單路線：AI Avatar）

  1. 登入 Kling，找 「AI Avatar」（也可能叫 数字人 / 对口型 / Lip Sync），選「圖片 + 音檔」模式。
  2. 上傳你的 角色圖。
  3. 聲音選 「上傳音檔」：用工具產腳本後做的口播語音檔（或在 Kling 貼逐字稿、選中文國語 TTS）。
  4. 比例有得選就選 直式 9:16；長度要放得下整段語音。
  5. 按 生成，等約 1 分鐘 → 下載 mp4。
  6. ⚠️ 下載前看角落有沒有 浮水印（免費版通常有）、確認朋友帳號 點數夠。

找不到就找「数字人 / 对口型 / Talking Avatar」。臉動得怪 = 換更正面、臉更大的角色圖。想「邊走邊講」是進階做法（先圖生影片再對口型），第一支先用站著講跑通整條鏈。

🔊 口播語音怎麼來

兩種都可：①（推薦，聲音每支一致）用免費的 edge-tts 把口播逐字稿轉成台灣中文語音檔，男聲學長 zh-TW-YunJhe / 女聲學姐 zh-TW-HsiaoChen。②直接用 Kling 內建中文 TTS。第一支現成語音與操作見專案的 Kling素材/ 資料夾。

🧠 人設 & 內容線（速查）

人設、內容線、字幕樣式可在 config.json 改；主題清單也在那裡（topics）。
