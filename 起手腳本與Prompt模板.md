# 起手腳本與 Prompt 模板

人設預設：**毒舌但中肯的職涯前輩**。格式：藏鏡人街訪（鏡頭後提問 Q、鏡頭前導師答 A，邊走邊講）。平台 IG Reels，15–40 秒。

---

## 一、腳本批量生成 Prompt 模板（丟給 LLM 一次出 10–20 集）

```
你是台灣職場短影音帳號的腳本編劇。帳號人設：一位「毒舌但中肯的職涯前輩」，
講話直接、會戳破職場迷思，但每句都站得住腳。格式是「藏鏡人街訪」：
鏡頭後一個看不見的提問者(Q)丟痛點問題，鏡頭前的前輩(A)邊走邊回答。

請產出 10 支 IG Reels 腳本，每支 15–40 秒，平均分布在這 4 條內容線：
1.求職硬技能 2.職場生存 3.迷思破解 4.情境模擬短劇

每支腳本格式固定：
- 標題（含內容線標籤）
- Hook（0–2秒，要嗆/反差/戳心，能讓人停下來）
- 本體（口語逐字稿，最多 3 個重點，講人話、不要條列腔）
- 收尾金句 + CTA（引導追蹤）
- 建議畫面（走在哪、什麼運鏡、字幕重點字）

限制：用台灣用語、口語、不說教；每支主題不重複；Hook 不要都用同一種句型。
```

> 換人設只改第一段；要 niche down（例如只做新鮮人）就在「限制」加一句鎖定受眾。
> 這個模板的程式化版本已內建在 `career-reels-factory/app/providers/prompts.py`。

## 二、起手腳本 ×8（可直接拿去 pilot）

### ① 求職硬技能
- **Hook（Q）**：「履歷投出去都石沉大海，是不是運氣不好？」
- **A**：「不是運氣，是你的履歷前三行沒人想看完。HR 平均看一份履歷 7 秒——把『最近、最強、跟這份工作最相關』的成果放最上面，數字講出來。『負責社群經營』沒人記得，『3 個月把粉專做到 1 萬』才會。」
- **收尾**：「履歷不是自傳，是廣告。追蹤我，下次教你寫出會被看完的開頭。」
- **畫面**：走在辦公大樓騎樓，手指比「7 秒」；字幕標「7 秒」「數字」

### ② 求職硬技能
- **Hook（Q）**：「面試最後問『有沒有問題要問』，隨便回沒關係吧？」
- **A**：「回『沒有』直接扣分。這題在測三件事：你做過功課沒、你在不在乎、你會不會思考。反問一句『這個職位過去半年最大的挑戰是什麼？』瞬間贏一半。」
- **收尾**：「面試是雙向選擇，不是被審問。」
- **畫面**：邊走邊扳手指數 1、2、3

### ③ 職場生存
- **Hook（Q）**：「主管把我的功勞講成他的，我要當場戳破嗎？」
- **A**：「當場撕破臉你輸。記住三招：留證據（重要的事用信件/訊息收尾）、用數據說話、向上管理而不是對抗。讓更上面的人『剛好』看到是你做的，比你大吼有用一百倍。」
- **收尾**：「職場不是比誰委屈，是比誰留得下證據。」
- **畫面**：走廊邊走邊講，字幕標「留證據」

### ④ 職場生存
- **Hook（A 反差開場）**：「想加薪卻開不了口的人，通常都犯同一個錯。」
- **A**：「你以為加薪是『拜託主管』，其實是『讓他無法拒絕』。別講『我很努力』，講『我這一年多扛了哪些事、帶來什麼結果，市場行情是多少』。把它變成一筆划算的投資，不是人情。」
- **收尾**：「會吵的孩子有糖吃？不，會算的孩子才有。」
- **畫面**：走進咖啡廳坐下，像準備談判

### ⑤ 迷思破解
- **Hook（Q）**：「大家都說年輕要『多嘗試』，是不是不該太早定方向？」
- **A**：「『多嘗試』被講爛了。亂試三年，履歷會變成一張沒重點的拼貼。真正該做的是『有方向地試』——選一個大方向，在裡面換角色、換產業，累積的東西才會疊加，不是歸零。」
- **收尾**：「探索不等於亂逛。」
- **畫面**：走在岔路口/十字路口

### ⑥ 迷思破解
- **Hook（A）**：「『做你熱愛的事就不累』，這句話害慘很多人。」
- **A**：「熱愛不會讓工作不累，它只是讓你願意撐過累的部分。把興趣當飯吃之前，先問自己：這件事『難看的那一面』你受得了嗎？受得了，才是真的適合。」
- **收尾**：「喜歡是濾鏡，受得了才是真的。」
- **畫面**：邊走邊苦笑

### ⑦ 情境模擬短劇
- **Hook（A）**：「被問『你最大的缺點是什麼』，這樣回直接出局 👇」
- **A**：先演 NG 版（皺眉假掰）「我啊…就是太追求完美了～」→ 切換正解「我以前不太敢拒絕別人，案子越接越爆。後來我學會先排優先序、跟對方對齊期待，現在反而交付更穩。」
- **收尾**：「重點不是沒缺點，是你有沒有在進步。」
- **畫面**：同一人左右分割演兩版（或轉身換表情）

### ⑧ 情境模擬短劇
- **Hook（A）**：「要離職時，最聰明的講法不是『我不爽』。」
- **A**：演示對話「主管，謝謝這段時間的栽培（先給台階），我決定往○○方向發展（講成長、不講抱怨），會把交接做完整（留好名聲）。」
- **收尾**：「離職要走得漂亮——這圈子很小。」
- **畫面**：走出辦公室、回頭微笑

## 三、腳本怎麼變成影片（接 Kling）
拿到「口播逐字稿」後：
- 可用工具內建/本機 **edge-tts** 把逐字稿轉成台灣中文語音檔（男聲 `zh-TW-YunJhe`＝學長、女聲 `zh-TW-HsiaoChen`＝學姐），或直接用 Kling 內建中文 TTS。
- 在 **Kling** 用「固定角色圖（ChatGPT 生）＋ 口播語音」做對口型 mp4 → 上傳回工具自動上字幕＋疊開場字卡＋轉 9:16。
- 第一支的現成素材與完整操作見 `Kling素材/`。
